1. run mysql server and import all tables in /sql directory
2. open server project in WebStorm
3. edit your database settings in server/db/properties.js
4. run "npm install" in WebStorm terminal (make sure its in server directory)
5. create private key and SSL certificate and copy them to server/cert (names: server.key, server.crt)
6. run server
7. open android project in Android Studio
8. edit your IP and port settings in com.tenantsproject.flatmates.model.rest.Properties
9. run client application
10. enjoy!
