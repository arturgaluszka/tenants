1. run mysql server and import all tables in /sql directory
2. open server project in WebStorm
3. edit your database settings in server/db/properties.js
4. run "npm install" in WebStorm terminal (make sure its in server directory)
5. run server
6. open android project in Android Studio
7. edit your IP and port settings in com.tenantsproject.flatmates.model.rest.Properties
8. run client application
9. enjoy!
